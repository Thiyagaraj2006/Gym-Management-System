# Gym Management System

Gym Management System developed using Python and MySQL for managing gym member details and records efficiently.

##  Technologies Used

* Python
* MySQL
* mysql-connector-python

##  Features

* Add Gym Members
* View Member Details
* Search Member Records
* Update Member Information
* Delete Member Records
* Membership Management
* MySQL Database Connectivity
* Console-Based User Interface

##  Modules Included

* Member Registration Module
* View Members Module
* Search Member Module
* Update Member Module
* Delete Member Module
* Membership Management Module

##  Database

* MySQL Database
* Gym Member Records Table
* CRUD Operations

##  How to Run

```bash id="ewm6pb"
pip install mysql-connector-python
python gym_management.py
```

##  Project Description

The Gym Management System is a database-driven application designed to manage gym member information efficiently. The system allows users to store, update, search, and delete member records using Python and MySQL. This project demonstrates database connectivity and record management through CRUD operations.
